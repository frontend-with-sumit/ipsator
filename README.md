# Assignment - Product listing page
A responsive UI design for the product listing page.

## Tech Stack
- React
- Next.js
- TypeScript
- TailwindCSS

## Running the project
- Clone the project
```
git clone https://github.com/frontend-with-sumit/ipsator.git
```

- Install the dependencies
```
npm install
```

- Start server
```
npm run dev
```
